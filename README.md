# :computer: Technologies

This project was made using the follow technologies:

- [NodeJS](https://nodejs.org/en/)
- [Express](https://expressjs.com/)
- [MongoDB](https://docs.mongodb.com/)
- [ReactJS](https://reactjs.org/)
